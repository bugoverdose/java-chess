# 체스 미션

## 구현할 기능 목록

- [ ] 체스판을 초기화한다.
    - 각 행은 아래부터 1~8행으로 구성된다.
    - 각 열은 좌측부터 a~h열로 구성된다.
    - 백색 진영은 아래쪽에, 흑색 진영은 위쪽에 위치한다.
    - 2행과 7행은 각각 8개의 폰(Pawn)들로 구성된다.
    - 1행과 8행은 각각 2개의 룩(Rook), 2개의 나이트(Knight), 2개의 비숍(Bishop), 퀸(Queen)과 킹(King)으로 구성된다.

```
♖♘♙♕♔♙♘♖ 8 (row 8)
♗♗♗♗♗♗♗♗ 7
........ 6
........ 5
........ 4
........ 3
♝♝♝♝♝♝♝♝ 2
♜♞♟♛♚♟♞♜ 1 (row 1)

abcdefgh
```
